📘 Client Integration Guide

Everything you need to integrate WhatsApp Sender API

🤔 What is WhatsApp Sender?

A powerful API platform that lets you send WhatsApp messages programmatically. You get an API Key, create WhatsApp sessions, and send messages to any number.

1️⃣ Get Your API Key

Your API Key is provided by the platform admin. Keep it secret and use it in all API requests.

# Example request:
curl -X POST https://your-server.com/v1/send \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"to":"201234567890","message":"Hello from API!"}'

2️⃣ Check Balance

Check your remaining message balance anytime:

curl -X GET https://your-server.com/v1/balance \
  -H "X-API-Key: YOUR_API_KEY"

# Response:
{"usedMessages":150,"maxMessages":1000,"remaining":850,"status":"active"}

3️⃣ Manage WhatsApp Sessions

You can create and manage your WhatsApp sessions via API or through the client panel.

# Create session:
curl -X POST https://your-server.com/v1/sessions/create \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"phone":"201234567890","name":"My Phone"}'

# List sessions:
curl -X GET https://your-server.com/v1/sessions \
  -H "X-API-Key: YOUR_API_KEY"

# Get QR code for scanning:
curl -X GET https://your-server.com/v1/sessions/{sessionId}/qr \
  -H "X-API-Key: YOUR_API_KEY"

4️⃣ Send Messages

Send messages through your connected WhatsApp session:

# Send a message:
curl -X POST https://your-server.com/v1/send \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"to":"201234567890","message":"Hello World!"}'

# Response:
{"success":true,"messageId":"abc123","status":"queued"}

💡 Pro Tips

  • Always use international format: 201234567890 (Egypt) or 966501234567 (Saudi Arabia)
  • Keep your session connected — scan QR again if disconnected
  • Each message counts toward your plan limit
  • Contact admin to upgrade your plan anytime
